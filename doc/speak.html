智能对话功能说明

On this page

💬 Ai心理学家 - 智能对话功能详细说明 ​

版本：v2.0
创建日期：2025-11-10

一、功能概述 ​

1.1 智能对话系统简介 ​

智能对话系统是哪吒小助手的核心功能，通过自然语言处理技术，实现与用户的智能交互。

核心能力：

  • 🤖 理解用户意图
  • 👤 个性化服务
  • 🗄️ 智能匹配
  • 💬 自然交互

1.2 核心功能模块 ​

智能对话系统
├── 身份管理（身份选择、切换、问候）
├── 意图识别（信息提取、问题识别、服务推荐）
├── 个案匹配（姓名查询、同名处理、详情展示）
└── 对话交互（文字输入、语音输入、消息展示）

二、身份管理系统 ​

2.1 支持的身份角色 ​

👨‍⚕️ 心理老师 ​

  • 核心职责：学生心理测评、专业咨询、记录跟踪
  • 可用功能：心理测评、案例检索、导师工具包、报告生成
  • 问候语：老师，[时段]好！

👨‍🏫 班主任 ​

  • 核心职责：了解学生状态、获取辅导建议、协同工作
  • 可用功能：心理测评（基础）、案例检索（查看）、导师工具包
  • 问候语：老师，[时段]好！

👔 校领导 ​

  • 核心职责：查看数据统计、制定工作计划、监督进展
  • 可用功能：数据统计、报告查看、案例检索（查看）
  • 问候语：领导，[时段]好！

2.2 身份选择流程 ​

首次使用：

打开应用 → 输入消息 → 弹出身份选择 → 选择身份 → 保存 → 开始使用

身份切换：

输入"切换身份" → 显示切换卡片 → 选择新身份 → 更新保存 → 切换成功

三、智能意图识别 ​

3.1 识别维度 ​

学生信息识别 ​

1. 姓名识别

  • 支持常见中文姓氏 + 1-3个字的名字
  • 示例：张三、李四、王小明

2. 年龄识别

  • 格式：数字 + 年龄单位
  • 示例：12岁、14周岁

3. 性别识别

  • 关键词：男/女/男生/女生/男孩/女孩
  • 示例：男生、女孩

问题识别 ​

问题分类：

  • 学习问题：压力、厌学、注意力
  • 情绪问题：焦虑、抑郁、烦躁
  • 睡眠问题：失眠、多梦、早醒
  • 人际问题：人际关系、孤独
  • 行为问题：叛逆、冲动、攻击
  • 自我认知：自信、自卑、价值

3.2 识别流程 ​

用户输入 → 提取信息 → 检查完整性 → 查询个案库 → 推荐服务 → 显示结果

3.3 使用示例 ​

完整信息输入：

输入：张三，12岁，女生，学习压力很大，晚上睡不着
识别：姓名=张三，年龄=12，性别=女，问题=学习压力+失眠
推荐：心理测评、案例检索、心理状态评估

不完整信息输入：

输入：张三，学习压力大
识别：姓名=张三，问题=学习压力
提示：请补充年龄和性别信息

四、个案库匹配 ​

4.1 查询流程 ​

输入姓名 → 精确匹配 → 返回结果
  ├─ 1个 → 直接显示详情
  ├─ 多个 → 显示选择列表
  └─ 0个 → 提示创建新档案

4.2 使用示例 ​

唯一匹配：

输入：李四
结果：显示李四的详细档案信息

多个匹配：

输入：张三
结果：显示2个同名学生的选择列表

未找到：

输入：王五
结果：提示未找到，建议创建新档案

五、对话交互 ​

5.1 输入方式 ​

文字输入：

  • 在输入框输入文字
  • 点击发送或按回车键
  • 支持多行文本

语音输入：

  • 点击切换到语音模式
  • 按住🎤按钮说话
  • 松开自动发送

5.2 消息展示 ​

  • 用户消息：右侧显示，蓝色背景
  • AI消息：左侧显示，白色背景
  • 卡片消息：结构化信息展示
  • 操作按钮：引导下一步操作

六、使用示例 ​

完整对话示例 ​

用户：你好
AI：老师，下午好！我是您的智能心理辅导助手。

用户：张三，12岁，女生，学习压力很大
AI：✅ 已识别学生信息
    姓名：张三，年龄：12岁，性别：女生
    问题：学习压力
    推荐：心理测评、案例检索

用户：切换身份
AI：显示身份切换卡片

用户：[选择班主任]
AI：✅ 身份切换成功！已切换为班主任
